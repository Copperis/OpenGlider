Installation
------------

install all dependencies first:
    * ezodf(2)
    * dxfwrite
    * scipy
    * (svgwrite)
    * (vtk)

clone the repo::

    git clone https://github.com/hiaselhans/OpenGlider.git


and install using setup.py::

    cd OpenGlider
    python2 setup.py install

(developers choice)::

    python2 setup.py develop
